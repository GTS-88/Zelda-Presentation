Plataforma y Control

Exclusividad Diseñado para Nintendo Switch y Wii U.
Mando Obligatorio Controles optimizados para el uso de sticks y botones físicos.
Emulación Posible en PC, pero requiere mando para una experiencia natural.

Interacción y Controles

Libertad Total Exploración no lineal de montañas, ríos y bosques.
Controles Contextuales Botones que se adaptan según si interactúas con un NPC, objeto o enemigo.
Puntería Uso de giroscopio para mayor velocidad y precisión con el arco.

Motor Químico

Sistémica Los elementos (fuego, agua, electricidad, frío) reaccionan entre sí de forma lógica.
Creatividad No se limita a apretar botones: fomenta el uso de objetos para resolver situaciones.

Piedra Sheikah

Runas Principales Bombas remotas, Magnesis (imán), Cryonis (hielo) y Stasis (paralizador).
Mecánica Disruptiva Permite romper las reglas tradicionales del juego mediante la lógica física.

Fluidez y Dificultad

Sin Cargas El mundo está siempre activo, manteniendo al jugador en inmersión constante.
Dificultad Justa La muerte suele ser consecuencia de una mala interacción con el entorno, no de los controles.
Switch vs Wii U Switch ofrece mejor resolución y vibración HD; Wii U inferior y menos fluidez.
